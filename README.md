# TaskSync

A Spring Boot backend for a task management application, developed as a university group project. The repository contains the server-side logic designed to handle business operations and data management for the TaskSync client application.

The frontend client for this application is available in a separate repository: [front_tasksync](https://github.com/nkalita497/front_tasksync).

### Requirements
* Java Development Kit (JDK) (version 17)
* Apache Maven (or IDE-integrated build tool)
* Java IDE (e.g., IntelliJ IDEA, Eclipse)

### Setup Instructions

1. Clone the repository:
`git clone https://github.com/Arthemisxx/TaskSync.git`

2. Navigate to the main project directory:
`cd TaskSync/demo`

3. Compile the project:
`mvn clean install`

4. Run the Spring Boot application. You can do this directly from your IDE by running the main application class, or via the terminal using:
`mvn spring-boot:run`

The application will typically start on the local server, accessible at http://localhost:8080 by default.
